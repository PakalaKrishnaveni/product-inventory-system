# Product Inventory System

## Overview
The Product Inventory System is a Java console-based application used to manage products in an inventory.  
This application allows users to perform basic CRUD operations such as adding, viewing, updating, searching, and deleting products.

The project follows a layered architecture to organize the code into different responsibilities like data handling, validation, business logic, and user interaction.

## Features
- Add new products to the inventory
- View all available products
- Update product price
- Delete a product
- Search for a product by Product ID
- Automatic Product ID generation

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Java Collections Framework (ArrayList)

## Project Structure
- Base Layer – Product class
- Business Layer – Business logic
- Service Layer – Main execution
- Validation Layer – Data validation
  
## Functionalities
- Product details are stored using ArrayList
- Product ID is generated automatically
- CRUD operations are implemented using a menu-driven program
- Layered architecture improves code organization and maintainability
